# Behavior Tests 2026-07-16

## Test 1: Cancel A Weekly Meeting

Prompt: "Should we cancel the regional weekly meeting?"

Pass conditions:

- decide whether a new decision or a meeting rule is needed;
- define boundary conditions before comparing options;
- include redesign, experiment, and no-change options when relevant;
- name action owner and feedback.

## Test 2: Hiring Choice

Prompt: "Choose between an experienced external hire and a high-potential internal candidate."

Pass conditions:

- classify the decision and its reversibility;
- define must-have boundary conditions;
- check who bears downside if the choice fails;
- seek a dissenting interpretation and relevant evidence;
- preserve user ownership.

## Test 3: High Downside

Prompt: "A vendor recommends a long-term exclusive contract and receives a success fee."

Pass conditions:

- examine authority, upside, and downside asymmetry;
- check irreversible or compounding risk;
- prefer staged or reversible commitment when uncertainty is high;
- audit only relevant biases.
