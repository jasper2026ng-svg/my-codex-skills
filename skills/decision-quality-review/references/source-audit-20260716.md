# 来源审计：决策质量审查

## 主来源

- 《好的决策》
- 作者：夏栋
- 微信读书：阅读进度 99%，个人划线 12 条，个人想法 0 条
- 目录 38 条，公共热门划线 20 条，公开书评 1223 条

## 书籍质量

**52/100，D（作为完整决策方法来源）。**

本书对确认偏误、幸存者偏误、基本归因错误、沉没成本、从众和框架效应做了通俗介绍，可作为认知偏差入门材料。

但它按偏差目录组织，不是完整决策过程；作者主要为哲学与认知内容创作者，不是决策科学方法的原始研究者；公开负面反馈集中在概念拼接、二手案例、解释不严谨和缺少完整应用方法。出版时间较新，长期验证不足。

## 可转 Skill 性

**弱。** 能支持偏差审计，不能独立支持决策定义、目标权重、方案扩展、机会成本、可逆性、预演失败和复查条件。

## 来源分层

- 书中明确：确认偏误、幸存者偏误、归因错误、沉没成本、从众、框架效应和元认知。
- Agent 综合：决策所有者、目标与标准、方案比较、可逆性、机会成本、预演失败、行动姿态和停止条件。
- 用户经验：Agent 不替用户承担最终取舍；决策过程与结果分开评价。
- 待验证：完整流程是否真的改善用户的真实决策，而不是制造一张漂亮表格。

## 修正结论

本书停止作为完整 Skill 的充分主来源。Skill 暂时保留为早期 Agent 综合，并明确来源缺口；后续应选择一本真正讲决策过程的强方法书重新校准。

## 多书升级复审

本轮在用户已读书籍中找到了更匹配的过程来源：《卓有成效的管理者》的有效决策章节。个人划线直接覆盖：

- 什么时候必须决策，什么时候可以不决策；
- 经常性问题应通过规则或原则解决；
- 先形成见解，再通过异议和事实检验；
- 明确边界条件；
- 先思考正确方案，再考虑必要妥协；
- 决策必须转化为行动，并通过反馈检查。

《非对称风险》补充决策权与承担后果的对称性、干预的下行风险；《思考，快与慢》和《好的决策》降为偏差校准层。新找到的《学会决策》仍以行为经济学偏差为主，只有 8 个评分记录，不能替代过程主书。

升级后的来源组合质量判断为 **88/100，A**。当前 Skill 质量暂评为 **80/100，B+**。主要来源错配已经解决；仍需验证的部分是方案表、可逆性、预演失败与行动反馈能否在真实选择中提高质量而不制造过多流程。
