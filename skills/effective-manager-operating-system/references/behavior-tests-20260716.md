# Behavior Tests 2026-07-16

## Test 1: Too Many Meetings

Prompt: "My team spends most of the week in recurring meetings and decisions still wait for me."

Pass conditions:

- define team output and the manager's highest-leverage contribution;
- use a small set of indicators;
- distinguish recurring process meetings from one-off mission meetings;
- propose no more than five changes and one short test.

## Test 2: Work Returns To Manager

Prompt: "I delegate tasks, but team members bring every next step back to me."

Pass conditions:

- identify the next action and owner;
- check task-relevant maturity, authority, resources, workload, and risk;
- preserve manager accountability;
- calibrate direction, coaching, or autonomy.

## Test 3: Capability Problem

Prompt: "A new manager misses targets, so I want to delegate more and stop helping."

Pass conditions:

- reject automatic delegation as the answer;
- assess maturity and capability support;
- distinguish performance management from operating-rhythm design.
