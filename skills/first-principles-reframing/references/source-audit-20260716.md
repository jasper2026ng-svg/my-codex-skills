# 来源审计：第一性原理重构

## 主来源

- 《半小时讲透第一性原理（轻思维）》
- 作者：樊中恺
- 微信读书：阅读进度 96%，个人划线 0 条，个人想法 0 条
- 全书约 1.2 万字，目录 14 条，公共热门划线 20 条，公开书评 660 条

## 书籍质量

**55/100，C。**

优点：主题集中，给出了识别假设、寻找基本事实、重新构建和使用边界的入门结构；公共评价总体偏正面。

风险：篇幅很短，作者与出版信息验证有限；案例主要为二手名人故事和生活类比；负面反馈集中在内容浅、实操弱、论证不足和疑似 AI 化表达。个人没有划线，不能声称 Skill 来自用户深度阅读。

## 可转 Skill 性

**部分。** 可支持触发条件和三个基本动作，但不能独立支撑事实分类、硬软约束、方案组合、权衡和实验设计。

## 来源分层

- 书中明确：识别隐藏假设、追问基本事实、从事实重新构建、不是所有问题都需要第一性原理。
- Agent 综合：事实类型、硬软约束、三个重构选项、权衡表和最小实验。
- 用户经验：第一性原理是条件分支，不是每次都用。
- 待验证：Agent 能否避免把主观判断包装成“基本事实”，以及是否会为了创新而创新。

## 修正结论

该书不再被表述为强主来源。Skill 保留为早期 Agent 综合工具，待找到更可靠、方法更完整的来源并完成新场景测试。

## 多书升级复审

本轮没有找到一本能够独立支撑完整“第一性原理重构 SOP”的高质量工具书，因此没有用另一本流行书简单替换原书，而是采用互补来源：

- 《系统之美》：用户阅读 99%、划线 7 条、670 个评分记录。作者与原版出版来源可信，目录和个人划线直接覆盖系统目标、要素与关联、反馈、延迟、非线性和有限理性。适合修复“拆完就以为得到真理”的还原论风险。
- 《表象与本质》：322 条目录记录、894 个评分记录；用户尚未阅读。该书理论深度较高，可用于提醒类比和范畴化是思考核心，但不适合直接转成操作步骤。

升级后的来源组合质量判断为 **82/100，A-**；可转 Skill 性仍为“部分”。当前 Skill 质量暂评为 **74/100，B-**。它已从轻量书衍生流程升级为“假设拆解 + 系统校准 + 最小试验”，但仍需证明这套组合不会因为检查项过多而拖慢普通工作问题。
