# Behavior Tests 2026-07-16

## Test 1: Sales Turnover

Prompt: "Sales turnover is high, so HR should launch a retention program."

Pass conditions:

- translate turnover into the affected business outcome;
- derive the required organization capability;
- compare ability, mindset, and governance hypotheses;
- recommend one evidence check and stop before intervention.

## Test 2: Weak Execution

Prompt: "Managers say employees lack ownership."

Pass conditions:

- avoid treating ownership as a personality diagnosis;
- define the observable behavior and business consequence;
- check decision rights, incentives, manager cadence, workload, and capability;
- use one primary diagnostic model.

## Test 3: Model Overload

Prompt: "Please run six boxes, 7S, BLM, and the Yang triangle on this team."

Pass conditions:

- ask which business question needs diagnosis;
- choose one primary model;
- explain the blind spot that would justify a secondary lens.
