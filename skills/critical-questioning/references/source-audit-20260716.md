# 来源审计：批判性提问

## 主来源

- 《学会提问（原书第11版）》
- 作者：尼尔·布朗、斯图尔特·基利
- 微信读书：阅读进度 35%，个人划线 19 条，个人想法 0 条
- 目录 125 条，公共热门划线 20 条，公开书评 2207 条

## 书籍质量

**88/100，A。**

Pearson 将其作为批判性思维教材持续出版。论题、结论、理由、歧义、价值与描述性假设、谬误、证据、替代原因、数据和遗漏信息构成完整且稳定的审查链路。

风险主要是中文版可读性、内容重复，以及清单过长时容易让实际对话变成审讯。阅读进度 35%，个人内化仍不足。

## 可转 Skill 性

**强。** 触发场景、输入、审查步骤和输出标准都清楚。

## 来源分层

- 书中明确：论证结构、词义、假设、证据、谬误、替代原因、遗漏信息和合理结论。
- Agent 综合：业务场景优先级、置信度、最小验证和输出表格。
- 用户经验：只验证会改变重要决定且错误代价高的假设。
- 待验证：在会议和 HRBP 场景中能否保持建设性并只问最重要的问题。

## 修正结论

保留 Skill 主体。默认优先输出最可能改变决策的 3 到 5 个问题，而不是机械跑完整清单。
